Gestion de versions

Bienvenue ! Chacune des 9 questions propose 3 versions du même problème, accessibles depuis les onglets au-dessus de l'énoncé. Ceci est la version facilemoyennedifficile de cette question.

Votre score pour chaque question est le meilleur de vos scores parmi les trois versions. Plus une version est difficile, plus elle rapporte de points.

Si vous trouvez que la version facile d'une question est très simple, vous pouvez donc la passer et résoudre directement la version moyenne ou même difficile pour gagner du temps.

Si par contre vous bloquez sur les versions moyennes ou difficiles de certaines questions, préférez résoudre les versions faciles des autres questions. Avec toutes les versions faciles résolues, vous aurez un peu plus de la moitié des points.

Vérifions que vous avez compris :

Résoudre la version difficile rapporte tous les points d'une question.
Résoudre la version facile de toutes les questions donne plus de la moitié des points.
On peut avoir tous les points d'une question sans résoudre la version moyenne.
