Fonction rectangle

Chaque exemple ci-dessous est formé d'une séquence d'instructions à exécuter dans l'ordre. Chaque instruction est un appel à la fonction rectangle, qui dessine un rectangle dans la grille située en bas à droite.

Les 5 nombres entre parenthèses sont les paramètres de la fonction rectangle, et décrivent, dans l'ordre :

sa colonne
de gauche		 sa ligne
du haut		 sa colonne
de droite		 sa ligne
du bas		 sa couleur :
0 pour du noir,
1 pour du blanc.

Complétez le programme ci-dessous pour qu'il reproduise le motif de gauche dans la grille de droite :

Notez que le programme ne peut pas contenir plus de 4 instructions.

Solution

  • Version facile :
    • rectangle(1, 2, 7, 3, 0)
    • rectangle(1, 2, 5, 5, 0)
  • Version moyenne :
    • rectangle(0, 0, 9, 6, 0)
    • rectangle(1, 1, 8, 5, 1)
    • rectangle(4, 0, 5, 6, 1)
    • rectangle(0, 2, 9, 4, 1)
  • Version difficile :
    • rectangle(1, 0, 9, 6, 0)
    • rectangle(3, 1, 7, 5, 1)
    • rectangle(4, 2, 9, 6, 1)
    • rectangle(3, 3, 5, 3, 0)
