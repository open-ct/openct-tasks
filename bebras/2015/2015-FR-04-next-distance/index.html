La case encadrée de rouge n'est pas marquée car elle est située à 3 pas de Castor et non 5.

C'est de l'informatique !

On peut généraliser le principe montré dans le sujet. On commence par marquer toutes les cases à distance 1 du point de départ, puis toutes les cases à distances 2, puis 3, etc. Une fois les cases marquées, on peut facilement trouver pour chaque case, comment atteindre le point de départ (à distance 0) le plus vite possible : il suffit de se diriger vers les plus petits numéros.
