Chez le dentiste

La semaine dernière, le dentiste a reçu 9 animaux et a noté les rendez-vous :

DateAnimalAdresseSoin
02/11/2012LoutreRivièreLimage
01/11/2012CastorForêtArrachage
01/11/2012CastorRivièreArrachage
01/11/2012LoutreRivièreNettoyage
30/10/2012LoutreForêtNettoyage
30/10/2012CastorRivièreLimage
29/10/2012CastorForêtArrachage
29/10/2012CastorBarrageArrachage
29/10/2012LoutreBarrageArrachage

Combien de castors ne venant pas du barrage sont venus pour un arrachage ?

La solution

On regarde les lignes du tableau une par une, et on compte le nombre de lignes telles que :

  • la colonne "Animal" contienne "Castor"
  • la colonne "Adresse" ne contienne pas "Barrage" (donc contienne "Rivière" ou "Forêt")
  • la colonne "Soin" contienne "Arrachage"

Les lignes 2, 3 et 7 conviennent. La bonne réponse est donc 3.

Remarque : le sujet présenté pendant le concours comportait une coquille; sur certaines lignes 2012 était remplacé par 2011, et du coup certains participants ont répondu 2 à la place de 3. Pour remédier à cette erreur, nous avons donné aux participants concernés les points associés à ce sujet. Le sujet est maintenant corrigé.

C'est de l'informatique

Le tableau présenté correspond à une version simple de ce que qu'on appelle en informatique une base de données. La question que l'on pose s'appelle une requête sur la base de données.

On peut tout à fait demander la réponse à cette requête à un ordinateur. Il suffit de lui poser la question dans un langage qu'il comprend. Il existe un certain nombre de langages utilisables. Le plus courant s'appele SQL (pour "Structured Query Language"). En SQL, la question du sujet serait programmée ainsi :

   SELECT COUNT(*) 
   FROM visites 
   WHERE Animal = 'Castor' 
   AND Adresse <> 'Barrage' 
   AND Soin = 'Arrachage'

Cette requête se lit de la manière suivante : compte le nombre de lignes dans la table des visites où le type de l'animal est égal à 'Castor' et l'adresse est différente de 'Barrage' et le type de soin est égal à 'Arrachage'. Les mots clés se traduisent ici ainsi : "SELECT" veut dire "récupère", "COUNT" veut dire "le nombre de", "WHERE" signifie "où", "AND" signifie "et", et le symbol "<>" signifie "différent de".

Culture informatique

Les bases de données sont extrêmement répandues dans les systèmes informatiques. Par exemple, le site du Castor utilise une base de données. Ainsi, à chaque fois que vous avez cliqué sur une réponse pendant le concours Castor, une requête a été exécutée pour enregistrer votre réponse dans la base de données du site Castor. De même, une autre requête a été utilisée pour calculer votre score total.
