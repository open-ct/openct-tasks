Code castor

Les Castors transportent sur la rivière des troncs de différentes tailles : Petit, Moyen, Long et Très long. Lorsqu'ils arrivent au barrage, ils décrivent leur chargement en jouant du tambour. Le tambour utilisé permet de jouer deux notes différentes, que l'on note "x" et "o". Pour décrire leur chargement, les Castors ont donc besoin d'associer une séquence de notes à chaque taille de tronc. Pour gagner du temps, les Castors ne veulent pas faire de pause entre les séquences de notes lorsqu'ils décrivent plusieurs troncs.

Avec cette contrainte, un seul des codes suivants peut être utilisé. Lequel ?

La solution

La musique faite par la trompe va être une longue suite de notes, sans pause entre elles. Il ne sera pas possible de s'appuyer sur des pauses sonores pour savoir où commence l'indication d'une taille et où elle finie. Il faut donc choisir le code de telle sorte que la suite de notes d'une taille ne peut être mal coupée et confondue avec une autre. Plus précisément, il faut que chaque code d'une taille ne commence pas par le code d'une autre taille. La proposition A ne fonctionne pas car le code de Moyen : xx débute par le code de Petit : x. Donc le message "xx" peut aussi bien correspondre à un chargement comportant un tronc moyen ou qu'à un chargement de deux petits troncs. Seule la proposition C propose des codes que l'on ne peut interpréter que d'une seule façon.
