Robot bâtisseur

Une robot placé sur une grille cherche à atteindre la case verte. Le robot est programmé pour avancer tout droit. Quand il atteint un obstacle (une case noire ou un bord de la grille), il tourne de 90° vers la droite et recommence à aller tout droit. Mais attention, chaque case par laquelle le robot est passé devient un nouvel obstacle !

Les flèches sur les cartes montrent les points de départ et les directions de départ du robot.

Sur quelle carte le robot n'atteint pas la case verte ?

La solution

Le robot n'atteint pas sa cible sur la carte C.

C'est de l'informatique
