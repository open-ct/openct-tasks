Deux billes

Écrivez un programme permettant de faire sortir les deux billes du labyrinthe. Chaque commande du programme fait rouler les billes dans une certaine direction.

Commencez par exécuter l'exemple ci-dessous pour bien comprendre le fonctionnement. Faites ensuite glisser les commandes afin de modifier le programme.
